# README #

# Installation
---
```
$ npm i -g herald-cli
```

# Getting Started
---

To initialize Herald and ensure you're environment is ready for deployment, run:

```
herald init
```

# Deploying to AWS
---
 Once Herald has been initialized, it is ready to be deployed to AWS. Run:

 ```
 herald deploy
 ```

 # Tearing down Herald
 ---

 To destroy any evidence of Herald from AWS, run:

 ```
 herald destroy
 ```
